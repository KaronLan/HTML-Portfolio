Hierarchical Matrix Multiplier Accelerator

Skills

  • Verilog HDL
  • VLSI Design
  • Synthesis & PnR

Tools

  • Synopsys Design Compiler
  • Cadence Innovus
  • ModelSim
  • Linux Shell

Description

Designed and implemented a hierarchical matrix multiplier accelerator from RTL to physical layout. The project involved completing a single-core datapath for vector-matrix multiplication, verifying correctness through gate-level simulation, and extending the architecture into a dual-core system with asynchronous communication and output normalization. Optimization included pipelining, clock gating, and memory double-buffering for improved performance and reduced power.

Reflection

This project taught me how to connect high-level algorithmic design with hardware implementation details. I learned how design decisions at the RTL level impact timing closure and power trade-offs in physical design. It also strengthened my debugging and verification skills, especially when analyzing gate-level timing behavior and handling asynchronous data transfer between clock domains.
